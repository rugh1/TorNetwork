# Tor Network Project

A project to set up and run a kinda Tor Network on localhost.

## how to setup:
#### 1.run directory.py 
#### 2.open the run_nodes.bat file
#### 3.run Echo server
#### 4.run Client and enter Echo server's ports
#### 5.have fun
